## 🎓 Gelerntes & Best Practices

### Technisch
✅ Statische Websites sind oft besser (Sicherheit, Speed)  
✅ CSS Variables machen Anpassungen easy  
✅ IntersectionObserver für elegante Scroll-Animations  
✅ SVG für Icons (kein Download nötig)  
✅ Mobile-First Design ist Standard, nicht Option  

### Rechtlich
✅ DSGVO = ernst nehmen (Datenschutz mandatory)  
✅ TMG § 5 = Impressum muss sichtbar sein  
✅ AGB = wichtig für Business-Klarheit  
✅ Rechtsberatung = 150-300€ investieren lohnt sich  

### Security
✅ Keine externen Abhängigkeiten = weniger Sicherheitslücken  
✅ Statische Seiten = schwer zu hacken  
✅ SSL/HTTPS = absolut notwendig  
✅ Backups = unterschätzt aber essentiell
